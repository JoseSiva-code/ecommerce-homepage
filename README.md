# Ecommerce Homepage

This project will be an ecommerce homepage built with HTML, CSS, and JavaScript.

The page will load product data from the Fake Store API and display the products dynamically in a responsive product grid. Each product card will show the product image, name, and an "Add to cart" button.

The cart action will be handled in a separate JavaScript file, following the project requirement to keep the cart logic outside the main script.
